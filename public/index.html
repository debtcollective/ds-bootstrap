Button styles

Buttons retain the same basic sizes, proportions, and type-names of Bootstrap buttons, but with the colors adjusted to relfect those of the Debt Syndicate.

For now, button interactivity (hover states, etc) should also be based on the Bootstrap defaults, though we might explore some more interesting microinteractions in the future.

Important stylistic subtleties are indicated in red

Standard Buttons

The type within standard buttons is Libre Franklin Medium

Primary Secondary Success Danger Warning Info Light Dark

Standard buttons all have their base color, but also universally have a 1px border of color #2b2b2b at 25% opacity

As rgba this would be rgba(43,43,43,.25)
